Projects

Smart Entry System

Tech Stack: HTML, CSS, JavaScript, Flask, SQLite, OpenCV

Smart Entry is a secure, digital visitor management system built to streamline and modernize entry processes in colleges and administrative offices. The system allows visitors to fill a form, verify their mobile number via OTP, receive a QR code for access, and have their face captured for security logging.

Designed for both speed and security, it replaces manual registers with a contactless, trackable, and verifiable digital flow. I led the project development, implemented the backend using Flask, handled real-time face capture with OpenCV, and managed team coordination and testing.
